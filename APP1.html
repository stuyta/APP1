Calculator

  • Not 4, but 5 Function
    1. First, in order to actually use the calculator, you have to first identify the buttons.
    2. Before you get into any calculation, make sure that you hit the "Clear" button, because only then will the calculator be reset and you can do whatever you want.
    3. On the middle left, you will see that the numbers are positioned for you.
    4. Click "1". You should see a "1" appear in the box.
    5. In the middle top right, you will see a dark green box where you have your plus, minus, multiply, and divide functions.
    6. Click a button. It will appear in the input box.
    7. Then click another number, and then hit the = button. You should see the result of the expression in the input box.
    8. You can use decimals in the calculator, by clicking the "." button
    9. Note: The % button is not a percent sign, it is a modulus. This means that the answer will be the remainder of x divided by y.
    10. You can string expressions together through parentheses.
  • Scientific Functions
    • Make sure that with these functions, you put the number or expression FIRST before putting these functions down, because it wraps the parentheses around the expression.
    • Sin(x) returns the sine function.
    • Cos(x) returns the cosine function.
    • Tan(x) returns the tangent function.
    • Asin(x) returns the arc sine function.
    • Acos(x) returns the arc cosine function.
    • Atan(x) returns the arc tangent function.
    • Log(x) returns the logarithmic function.
    • √(x) returns the square root function.
    • Exp(x) returns the function e^x power.
    • Abs gives the absolute value of an expression.
    • Ceil rounds up the number.
    • Flr rounds down the number.
    • E gives the constant e.
    • LN2 gives the logarithm base 2 of E.
    • LN10 gives the logarithm base 10 of E.
    • π gives the constant pi.

URL

  1. You will see an input box that says, "Replace me with the URL you want to see."
  2. There are two buttons underneath it, one that says, "Delete iframe" and the other which says, "Clear". Beneath that there is a button that says "Submit".
  3. Type in any URL you want (make sure it has the "http://" in front of the site name.
  4. The Submit button displays an "iframe" which displays the website. The Clear button will empty out the input box. The Delete button will clear the iframe.

CSS Settings

  • NOTE: you must have some knowledge of CSS before continuing, otherwise the layout can change unpredictably.
  • For example, in the typical CSS syntax, you have: ".selector{propertytochange: parameter}"
  • In the first input box, type in the element needed to be altered.
  • The second box is where you type in the parameter of the element to be changed.
  • The third box is where you type in the new value.

If something goes wrong, click the "R" button to refresh or the "X" button to exit.
